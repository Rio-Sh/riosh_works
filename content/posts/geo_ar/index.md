**Geo AR message app**

{{< video src="geo_ar_demo" height=400px" >}}
  
### 概要
地図上にメッセージを配置し、ARモデルで可視化します。  
  
### github repository
[Rio-Sh/Mareaven](https://github.com/Rio-Sh/Mareave)  
※ビルドする場合local.propertiesでMAPS_API_KEY="{your_api_key}"の形でGoogle MapのAPIキーを入力して下さい。  
  
### 使用ライブラリ
* AR Core
* OpenGL ES
* Maps SDK for Android
